# ReportForge — 政府公文报告格式化工具

> **永久免费 · 禁止商用 · CC BY-NC-SA 4.0**

一键将 Word 文档格式化为符合政府公文标准的报告格式。

---

## ✨ 功能

- **智能格式化**：自动识别粘贴内容的标题层级，一键映射到报告规范
- **9级标题体系**：黑体三号→楷体三号→仿宋四号…自动编号
- **表格标准化**：外框1.5磅、内框0.5磅、黑体小四表头、仿宋小四表体
- **图片自动居中**：检测图片段落自动应用居中格式
- **页面一键设置**：A4 / 上下3cm / 左右2.6cm / 页眉1.5cm / 页脚2cm
- **目录自动生成**：规范三级目录，带页码超链接
- **配置可定制**：编辑 JSON 文件即可调整字体、字号、行距

## 📦 系统要求

- Windows 10 / 11
- Microsoft Office 2016+ (x64)
- .NET Framework 4.7.2+（Windows 10 自带）

## 🚀 安装

1. 下载并解压 zip 包
2. 双击 `安装.bat`（自动请求管理员权限）
3. 打开 Word → 看到「报告格式化」标签 → 安装成功

## 📖 使用

详见 `docs/使用指南.md`

## ⚙️ 自定义配置

在 Word 中点击 **报告格式化 → 打开配置文件**，用记事本编辑 `default.json`：

```json
{
  "bodyStyle": {
    "fontZh": "仿宋_GB2312",    ← 正文字体
    "fontSizePt": 14,            ← 四号
    "lineSpacingValue": 30       ← 固定值30磅
  },
  "headings": [
    { "level": 1, "fontZh": "黑体", "fontSizePt": 16 },
    { "level": 2, "fontZh": "楷体", "fontSizePt": 16 }
  ]
}
```

修改后点 **重载配置** 即可生效。
